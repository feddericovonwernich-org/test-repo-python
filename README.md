# Test Repo Python

A Python project demonstrating proper Python project structure and testing practices.

## Description

This repository tests the scorecards system with a Python-specific setup, including:
- pytest test framework
- Proper package structure
- Python-specific CI configuration
- Requirements management

## Installation

```bash
pip install -r requirements.txt
```

## Testing

```bash
pytest tests/
```

## Project Structure

- `mypackage/` - Main package
- `tests/` - pytest test suite
- `requirements.txt` - Python dependencies
